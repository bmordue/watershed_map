# Palette's Journal - Aberdeenshire Watershed Map

## 2026-02-12 - Initial Accessibility Assessment
**Learning:** The interactive map relies solely on mouse interactions, making it inaccessible to keyboard and screen reader users. Lack of focus indicators and ARIA roles means the interactive regions are "invisible" to assistive technologies.
**Action:** Implement ARIA roles, tabindex, and high-visibility focus-visible styles to ensure all users can explore the watershed data.
